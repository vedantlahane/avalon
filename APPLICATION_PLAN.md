# NexusCRM AI - Application Plan

## Overview
NexusCRM AI is an AI-First CRM designed for high-growth teams. It features a modern, minimal UI inspired by Linear and Notion, with deep AI integration for lead scoring, deal analysis, and inbox management.

## Core Features
1. **Intelligent Lead Scoring**: Automatically prioritize contacts based on interaction frequency and deal potential.
2. **Visual Deal Pipeline**: Drag-and-drop kanban board to manage sales stages and track revenue goals.
3. **Unified Inbox & Activity Feed**: Consolidate emails and tasks into a single stream.
4. **Automated Reporting**: Real-time insights on sales performance and productivity.
5. **AI Assistant**: A collapsible right panel providing context-aware suggestions and summaries.

## Tech Stack
- **Frontend**: React, TypeScript, Tailwind CSS v4, Lucide Icons, Framer Motion
- **State Management**: React Hooks + Context API
- **Mocking**: Local storage based mock services
- **AI Integration**: Agent SDK for sync/async event triggers

## Design System
- **Colors**: Indigo/Violet primary, Dark Slate sidebar, Light Gray background.
- **Typography**: Inter (Headings: Bold, Body: Regular).
- **Layout**: Collapsible left sidebar, toggleable right AI assistant.
